## Блок 14 — Безопасность API

## Автоматический прогон

```bash
npm run dev       # запустить сервер
npm run lint      # проверка без ошибок
npm run db:seed   # заполнить тестовые аккаунты
npm run db:backup # создать дамп перед изменениями
```
