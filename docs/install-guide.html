Magnifique

Инструкция по установке приложения

Адрес: magnifique-app.ru — откройте в браузере на вашем устройстве

🍎 iPhone / iPad

  1. Откройте Safari (только Safari)
  2. Перейдите на magnifique-app.ru
  3. Нажмите «Поделиться» внизу — квадрат со стрелкой ↑
  4. Выберите «На экран "Домой"»
  5. Нажмите «Добавить»
Значок появится на главном экране.

🤖 Android

  1. Откройте Chrome
  2. Перейдите на magnifique-app.ru
  3. Нажмите ⋮ (три точки) в правом верхнем углу
  4. Выберите «Добавить на главный экран» или «Установить приложение»
  5. Нажмите «Установить»
Значок появится на главном экране.

💻 Компьютер (Windows / Mac)

  1. Откройте Google Chrome или Microsoft Edge
  2. Перейдите на magnifique-app.ru
Приложение откроется в браузере.

🔑 Вход в приложение

  1. Введите ваш номер телефона
  2. Введите пароль, который выдал менеджер
  3. Нажмите «Войти»
Проблемы со входом? Обратитесь к менеджеру.
